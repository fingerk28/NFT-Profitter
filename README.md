# NFT Profitter

![](https://github.com/fingerk28/NFT-Profitter/blob/master/pic/poster.gif)

### Functionality

1. NFT trade information summary
   * Profit
     * Unrealized Profit
     * Realized Profit
     * Total Profit
   * Total Gas Cost
   * Winning Percentage
2. Whale tracking

### Preparation

```sh
pip install etherscan-python
pip install web3
pip install flask
```

### Usage 

```sh
python app.py
```

> Click the url and open in browser, and you'll see the homepage
